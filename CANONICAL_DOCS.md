# Canonical Docs

이 문서는 `Emptines`, `EM`, `UGC` 사이의 원본 소유권을 고정하기 위한 기준 문서다.

빠른 시작:
- 먼저 `docs/INDEX.md`를 읽는다.
- 그 다음 이 문서로 정본 소유권을 확인한다.

## 역할 구분

- `EM`: 공식 사이트이자 백서/세계관 정본 저장소
- `UGC`: 오브젝트 배치, 유저 흔적, 포탈 해석 등 UGC 운영 기준의 정본 저장소
- `Emptines`: 공식 사이트의 런타임 구현 저장소

## 이 저장소의 로컬 문서 취급

- `WHITEPAPER.md`: `EM` 정본의 미러
- `WORLD_LORE.md`: `EM` 정본의 미러
- `README.md`: 런타임/배포 설명의 원본

## 운영 원칙

- 백서와 세계관 수정은 가능하면 `EM`에서 먼저 한다.
- UGC 규칙 수정은 가능하면 `UGC`에서 먼저 한다.
- 이 저장소는 구현 문구와 런타임 안내를 맡되, 상위 철학을 재정의하지 않는다.
- 포탈 이동, 닉네임 연속성, 공 구역 UX는 장기적으로 백서 기준에 맞춰 수렴시킨다.

## 읽기 경고

- `docs/ugc`만 단독으로 읽고 전체 방향을 결정하지 않는다.
- `README.md`만으로 철학과 세계관을 해석하지 않는다.
- 문서 충돌 시 구현 편의보다 `EM` 정본 정의를 우선한다.
